üìã DETAILED INSTRUCTIONS:

BEFORE YOU START:
-----------------
You need Node.js installed on your computer.

Check if you have it:
  1. Press Windows Key + R
  2. Type: cmd
  3. Press Enter
  4. Type: node --version
  5. If you see a version number, you're good!

If not installed:
  ‚Üí Download from: https://nodejs.org/
  ‚Üí Install the LTS version
  ‚Üí Restart your computer

FIREBASE SETUP:
--------------
You need a Firebase project:

  1. Go to: https://console.firebase.google.com
  2. Click "Create a project"
  3. Enter project name: my-finance-app
  4. Continue through setup (disable Analytics if you want)

  5. Enable Authentication:
     - Click "Authentication" in left menu
     - Click "Get started"
     - Click "Anonymous" ‚Üí Enable ‚Üí Save

  6. Enable Firestore Database:
     - Click "Firestore Database" in left menu
     - Click "Create database"
     - Start in production mode
     - Choose a location close to you
     - Wait for it to create

     - Click "Rules" tab
     - Replace with:
       rules_version = '2';
       service cloud.firestore {
         match /databases/{database}/documents {
           match /users/{userId}/{document=**} {
             allow read, write: if request.auth != null && request.auth.uid == userId;
           }
         }
       }
     - Click "Publish"

  7. Get your config:
     - Click the gear icon ‚öôÔ∏è ‚Üí Project settings
     - Scroll to "Your apps"
     - Click the web icon </>
     - Enter nickname: My Finance App
     - Click "Register app"
     - Copy the firebaseConfig values

  8. Add to .env file:
     - Open .env with Notepad
     - Replace the values with your Firebase config
     - Save

RUNNING THE APP:
---------------

  Method 1 (Easy):
    ‚Üí Double-click START-APP.bat
    ‚Üí Browser opens automatically at http://localhost:3000

  Method 2 (Manual):
    1. Open Command Prompt in this folder
    2. Type: npm start
    3. Press Enter
    4. Browser opens automatically

STOPPING THE APP:
----------------

  In the Command Prompt window:
    ‚Üí Press Ctrl + C
    ‚Üí Type: Y
    ‚Üí Press Enter

USING THE APP:

  Add Income:
    ‚Üí Click the big + button
    ‚Üí Fill in client name, amount, date
    ‚Üí Click "Save Income"

  Add Expense:
    ‚Üí Click "Expense" button at bottom
    ‚Üí Fill in details
    ‚Üí Click "Save Expense"

  Recurring Bills:
    ‚Üí Click "Bills" button
    ‚Üí Add recurring bills
    ‚Üí Mark them as paid each month

TROUBLESHOOTING:
---------------

  White screen or errors:
    1. Press F12 in browser
    2. Look for red errors in console
    3. Check .env file has correct values
    4. Restart: Ctrl+C then npm start

  "npm is not recognized":
    ‚Üí Node.js not installed
    ‚Üí Install from https://nodejs.org/
    ‚Üí Restart computer

  Port 3000 in use:
    ‚Üí Close any other apps using port 3000
    ‚Üí Or use: set PORT=3001 && npm start
